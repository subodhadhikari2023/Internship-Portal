Instructions for creating an Internship

  • Enter the internship name (e.g., Rust Developer Intern).
  • Select the start and end dates for the internship.
  • Specify the required educational qualifications (e.g., Bachelor's in Computer Science).
  • Set the status of the internship (Active or Inactive).
  • Add the required skills (e.g., Java, Spring Boot, SQL).
  • Add the skills as comma separated values as described above.
  • Choose the work mode (Remote or On-Site).
  • Provide a detailed description, including roles and responsibilities.
  • Review all the details to ensure accuracy.
  • Click "Submit" to post the internship.
